## 解法

### 概要

$a = 999 \dots 9 \le n$ で最大のものを使えばよい。
$O(\log n)$。

## メモ

-   落として冷えた人がいくらかいた
-   証明分からない

## 実装

``` python
#!/usr/bin/env python3
def s(x):
    return sum(map(int, str(x)))
n = int(input())
a = 0
while a * 10 + 9 <= n:
    a = a * 10 + 9
b = n - a
print(s(a) + s(b))
```
