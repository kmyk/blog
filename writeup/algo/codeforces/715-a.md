## solution

According to the statement, when you push the $\sqrt{x}$ button, $x = m^2k^2(k+1)^2$ holds for some $m \ge 1$.
Fix $m = 1$, it's ok. Do it.

## implementation

``` python
#!/usr/bin/env python3
n = int(input())
x, k = 2, 1
while k <= n:
    y = ((k+1) * k) ** 2
    print((y - x) // k)
    x, k = (k+1) * k, k + 1
```
