## problem

本のページ番号が与えられるので、そのページ番号を含む最小のページの集合の枚数を答える。
$1$あるいは$2$あるいはその両方が与えられていたら表がp$1$で裏がp$2$な紙が$1$枚。

## implementation

``` python
#!/usr/bin/env python3
import random
n = int(input())
ps = set(map(int, input().split()))
result = len(ps)
for p in ps:
    if p % 2 == 1 and p + 1 in ps:
        result -= 1
print(result)
```
