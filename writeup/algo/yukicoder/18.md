# Yukicoder No.18 うーさー暗号

## [No.18 うーさー暗号](http://yukicoder.me/problems/59)

brainfuck向きの問題。
ひとつ気を付ける点があって、何文字目であるかのカウンタをoverflowさせるとwaが生える。

``` brainfuck
>>>>>
,----------[ while not newline
    >+++++++++[<------>-]< a is one
    <<<
        + inc counter
        <<+++++[>+++++<-]>+> set 26
        [>+<-]>[<+<->>-]< calc difference between 26
        <<+>[<[-]>-]> negate
        <<[- if overflow
            >>[-]<< set 0
        ]>>
        [>+>+<<-]>[<+>-]>> dup counter
    <[ counter times do
        >- decr
        [>+>+<<-]>[<+>-]+>[<[-]>-]<< calc is zero
        >[- if underflow
            +++++[<+++++>-]<+> set z 26
        ]<
        <
    -]>
    >++++++++[<++++++++>-]< add a
    .
,----------]
++++++++++.
```
