## [No.333 門松列を数え上げ](http://yukicoder.me/problems/935)

``` python
#!/usr/bin/env python3
a, b = map(int,input().split())
if a < b:
    print(b - 2)
elif a > b:
    print(2000000000 - 1 - b)
```
