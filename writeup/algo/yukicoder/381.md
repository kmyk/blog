## solution

単純には、$2N - (N + \lfloor \frac{N}{2} \rfloor + \dots + 0)$が答え。
これらを$2$進数で書くと、単に`popcount`すればよいことに気付ける。
$N$は$2^k$の整数に分解してそれぞれ考えて構わないことを見ると理解しやすい。

## implementation

``` python
#!/usr/bin/env python3
print(bin(int(input())).count('1'))
```
