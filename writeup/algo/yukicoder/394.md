# Yukicoder No.394 ハーフパイプ(1)

sortして真ん中$4$つの平均を取ればよい。
やるだけだけど、後の問題の布石になってるのは良いですね

``` python
#!/usr/bin/env python3
print('%.2f'%(sum(sorted(map(int,input().split()))[1:5])/4))
```
