# Yukicoder No.396 クラス替え

それぞれ何組に入るかは余り求めれば求まる。
丁寧にやる。後輩氏が実装ミスってWAを生やしてた。

``` python
#!/usr/bin/env python3
n, m = map(int,input().split())
x, y = map(int,input().split())
def f(i):
    i = (i-1) % (2*m)
    if i >= m:
        i = (2*m-i-1) % m
    return i
print(f(x) == f(y) and 'YES' or 'NO')
```
