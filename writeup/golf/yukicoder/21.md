# Yukicoder No.21 平均の差

最小化でなくて最大化であることに注意。誤読しないように。

## solution

$K \ge 3$なので最大値と最小値の差。$O(N)$。

## implementation

perl $56$byte。

``` perl
<>;<>;$a=$b=<>;$_<$a?$a=$_:$b<$_?$b=$_:0for<>;print$b-$a
```
