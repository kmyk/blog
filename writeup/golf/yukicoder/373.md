### python

``` python
#!/usr/bin/env python3
a, b, c, d = map(int,input().split())
print(a * b * c % d)
```

### perl 45byte

``` perl
($a,$b,$c,$d)=split$",<>;print $a*$b%$d*$c%$d
```

### tailsさん perl 35byte

<http://yukicoder.me/submissions/95737>

``` perl
use bigint;$/=$";print<>*1*<>*<>%<>
```

`$/=$"`は区切り文字を改行から空白にして、行ごとでなく単語ごとに読むようにする。
