# Yukicoder No.418 ミンミンゼミ

`\x0b` (`\v`, `^K`) はvertical tab。`bash`と`wc`で空白かどうかの認識がずれている。こういうの楽しい。

``` sh
$ cat a.sh | xxd
00000000: 7472 206e 200b 7c77 6320 2d77 0a         tr n .|wc -w.
```
