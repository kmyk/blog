# Yukicoder No.442 和と積

bash $37$byte。入力をbashで取ってrubyへ渡すテクは先日%20さんが用いていたもの。
tailsさんがこれにさらに`$[a+b]`とbash/rubyでの空白性の差を用いた$34$byteを提出していた。

``` sh
read a b;ruby -e"p ($a+$b).gcd $a*$b"
```
