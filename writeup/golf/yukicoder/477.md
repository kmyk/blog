## solution

$n/(k+1)+1$。

気にすべきは救援に$K$人来て全員が同程度のダメージを与えたとき。
つまり$\mathrm{ans} \gt \lfloor \frac{N - \mathrm{ans}}{K} \rfloor$であればよい。
この床関数は無視できて、変形すると$\mathrm{ans} \gt \lfloor \frac{N - \mathrm{ans}}{K} \rfloor$。

## implementation

bash $12$byte。自明だし当然提出速度負けしていた。

``` sh
dc -e?1+/1+p
```
