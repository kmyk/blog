# Yukicoder No.560 ふしぎなナップサック

## solution

$M \gets \frac{2M}{3} + \frac{M + 1}{3} + \frac{0}{3}$をするのは$\frac{1}{3}$増やすのと同じなので$M + \frac{N}{3}$。$O(1)$。

## implementation

``` sh
dc -e9k?3/+p
```

`$0+=$2/3`を考えたがawkは対応してないし出力精度も足りなかった。
